# Interview Prep Bot

AI-powered technical interview preparation bot for Slack.

## What It Does

Interview Prep Bot helps developers prepare for technical interviews at top companies by:

- 🤖 Simulating real coding interviews
- 🔍 Providing company-specific questions
- 📊 Tracking progress over time
- 💡 Offering intelligent AI feedback
- 🎯 Personalizing difficulty progression

## Technologies Used

- **Slack AI Capabilities** - Conversational interview flow & code evaluation
- **Real-Time Search API** - Latest interview questions from top companies
- **MCP Server Integration** - Code execution and testing
- **Node.js & Bolt SDK** - Core application framework

## Features

### 1. Personalized Interview Prep

- Select company, role, and experience level
- Get tailored questions matching your profile

### 2. AI-Powered Code Review

- Submit solutions directly in Slack
- Receive instant feedback on code quality
- Learn optimal solutions

### 3. Progress Tracking

- View questions solved and success rate
- Identify strong areas and topics to practice
- Track improvement over time

### 4. Intelligent Hints

- Get hints when stuck
- Learn problem-solving strategies
- Build confidence gradually

## Installation

1. Clone this repository
2. Run `npm install`
3. Create `.env` file with your Slack tokens
4. Run `npm start`
